    Adding Top Hat Student Registration link (LTI) to Canvas course

    Instructors can add the TopHat student registration link to a module in their Canvas course(s). To add the registration link, please follow the instructions below.

    To add the TopHat student registration link to new Module:

    1. In your course, click on Modules, then click on +Module.
    2. Give it the name 'TopHat Registration' (or name of choice).
    3. Click Add Module.
    4. In the appropriate module, click the + icon in the header to add an item to the module; select External Tool from the dropdown menu.
    5. A list of external tools will populate. Scroll down and select Top Hat from the list. Check the box next to 'Load in a new tab', then click Add Item.
    6. Click the Publish icon to the right of the LTI tool link.
    7. If the module is not yet published, click the Publish icon to the right of the module title.
    8. Click the link for the app to test the integration.

    If you already have a module created, you can skip steps 1-3.
