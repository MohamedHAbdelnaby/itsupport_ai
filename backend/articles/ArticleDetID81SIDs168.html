    OU Security Camera Software Update â€“ Avigilon Control Center

    The University of Oklahoma has updated the security camera software for the new Avigilon Control Center Client. This new software will change the way departments check their security camera feeds. To continue to access ones feeds, users will need to install the latest client on their security camera monitoring PC. Note the client only currently supports installation on the Windows operating system.

    To install, please visit the Avigilon site to download the latest client (Motorola account required).

    1. After downloading the client, double click the .exe file to begin setup.
    2. Your installer will then begin the install process.
    3. Uncheck the â€˜All users of this computerâ€™ radial button to prevent the software from opening upon computer startup (see below).
    4. If you receive a prompt that the Control Center Client needs to be closed, click Next to close it and continue.
    5. If Direct X 9.0c is not found on your computer, the client will then install this component.
    6. Once it is finished installing Direct X, click Next.
    7. Check the box to accept the terms of the license agreement and then click Next.
    8. Choose the install path folder and then click Next.
    9. Select your start menu options and then click Next.
    10. Now click Next and the client will install.
    11. Once complete, click Finish to complete the install process.

    Training videos for using the client and monitoring video are available through the link below:

    https://partners.avigilon.com/avigilonstore (Motorola account required)
