    Respondus Overview and Support

    Overview

    Respondus is a powerful tool for creating and managing exams that can be printed to paper or published directly to Blackboard, Canvas, and other learning systems. Exams can be created offline in a Windows environment, or moved from one LMS to another.

    Respondus offers several options for support for its products which include:

    1. Demo videos, which include how-tos on creating questions, moving exams, and publishing to various learning systems, can be found here: http://www.respondus.com/products/respondus/demos.shtml
    2. Respondus Support is available here: https://support.respondus.com/support/
    3. A support ticket to Respondus can be submitted here.
    4. Respondus Email Support is available through support@respondus.com.

    Please note that while OU IT can provide general support regarding setting up Respondus in Canvas, questions beyond the initial technical setup of Respondus in a course should be handled by Respondus Support directly.
