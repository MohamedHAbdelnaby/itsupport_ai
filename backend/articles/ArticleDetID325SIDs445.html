    Connecting Zoom Add-in to Outlook Web or Desktop Application

     

    Connecting Zoom Add-in to Outlook Web Application

    WINDOWS USER

    Log in to the Outlook web app.

    1. Open the Add-ins menu
      • Select a message
      • Select More Actions (three horizontal dots) at the top-right of the message
      • At the bottom of the list, select Get Add-ins

         
    2. Once in the Add-ins menu, type “Zoom for Outlook” in the search bar. Click Add
    3. Zoom calendar functionalities will now be available in Outlook.

    MAC USER

    1. Log in to the Outlook web app.
    2. Open the Add-ins menu
      a. ​In the Options sidebar, under the General Tab, click on Manage add-ins

    1. You will be directed to Outlook’s browser Add-ins Manager.
      a. Click on the plus icon > Add from the Office Store
    2. Once in the Add-ins menu, type “Zoom” in the search bar. Click on Get it now
    3. Zoom calendar functionalities will now be available in Outlook.

     

    Connecting Zoom plugin to Outlook Desktop Application

    NOTE: TO INSTALL ZOOM FOR OUTLOOK ON THE OUTLOOK DESKTOP APP, YOU MUST HAVE A MICROSOFT EXCHANGE ACCOUNT ADDED TO OUTLOOK.

    WINDOWS USER

    1. Open the Outlook desktop app and, for Windows users, go to File > Manage Add-ins.
    2. You will be directed to Outlook’s browser Add-ins Manager.
      • Click on the plus icon > Add from the Office Store.
    3. You will be re-directed to the Office Store. Search for Zoom for Outlook and click GET IT NOW
    4. Zoom calendar functionalities will now be available in Outlook.

    MAC USER

    1. Go to click the Store icon on the Outlook ribbon
    2. You will be re-directed to the Office Store. Search for Zoom and click GET IT NOW
    3. Zoom calendar functionalities will now be available in Outlook.
